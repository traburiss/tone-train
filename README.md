# 音乐助教 (Music Teaching Assistant)

原本是一个基于 Web 的音乐听音与音阶练习工具，并且我一直没时间搞这个项目，所以基础功能搞定满足需求后，就一直没动力继续项目。

现在大模型的能力越发成熟，我尝试了几天 AI 驱动开发之后，决定继续这个项目，毕竟现在我可以在练琴的同时（让AI）搞开发，练琴间歇 CR 和功能测试就可以把功能搞定。

同时由于我个人的实际音乐学习需求，所以我重新设定了整个项目的目标，准备加入更多的功能功能，现已升级为辅助音乐学习的综合性助教工具。

目标是协助用户进行全方位的音乐训练，涵盖器乐（主要是吉他）与声乐等多个维度。

🔗 **在线体验**: [https://traburiss.github.io/music-teaching-assistant/](https://traburiss.github.io/music-teaching-assistant/)

---

## ✨ 核心功能与规划

### � 器乐训练（以吉他为主）

#### 1. 听音训练 (已初步实现 ✅)

目前已有的核心功能，用于提高听感。

- **音阶/和弦播放**: 自动循环播放音阶或和弦，并语音播报音名。
- **多种模式**: 支持单音练习、吉他和弦练习。
- **高度定制**: 支持选择练习范围、音色（原声吉他、电吉他、钢琴）、播放速度等。
- **语音辅助**: 支持播报音名、唱名（Do, Re, Mi...）及简谱数字。
- **主音引导**: 支持“先弹主音，再弹目标音”模式。

#### 2. 听音判断 (计划中 �)

给一个音或者和弦，并提供数个带选项，从带选项中选出正确的音。

- **玩法**: 选错就一直选，直到选对为止。
- **难度分级**: 支持不同难度层级，不同难度要求选择不同数量的学习音数，待选项也会不同。
- **数据统计**: 支持统计不同难度听音正确率变化和判断耗时变化。

#### 3. 听音画和弦 (计划中 🚧)

给一个和弦，要求用户画出（或选择）对应的和弦指法图。

- **数据统计**: 支持统计不同难度的正确率变化和耗时变化。

#### 4. 辅助工具

- [ ] **节拍器** (待定)
- [ ] **调音器** (待定)

### 🎤 声乐训练 (计划中 🚧)

#### 1. 发声练习

只唱 `ma` 音，从指定音阶一路下降。

- **练习范围**: 支持设定默认练习范围（男声/女声），并允许用户手动选择。
- **引导方式**: 播放引导音与需要演唱的音阶。

#### 2. 元音练习

例如 `mimami` 练习。

- **练习范围**: 支持设定默认练习范围（男声/女声），并允许用户手动选择。
- **引导方式**: 播放引导音与需要演唱的音阶。

#### 3. 音域拓展练习

只唱 `E` 音，从指定音阶一路上升。

- **练习范围**: 支持设定默认练习范围（男声/女声），并允许用户手动选择。
- **引导方式**: 播放引导音与需要演唱的音阶。

## 🚀 快速开始

### 本地运行

1. **克隆项目**

   ```bash
   git clone https://github.com/traburiss/music-teaching-assistant.git
   cd music-teaching-assistant
   ```

2. **安装依赖**

   ```bash
   npm install
   # 或者
   pnpm install
   ```

3. **启动开发服务器**

   ```bash
   npm run dev
   ```

   打开浏览器访问 `http://localhost:8000`。

### 构建部署

```bash
npm run deploy
```
